# CLAUDE.md

このファイルは Claude Code がプロジェクトを読み込む際に自動的に参照されます。
プロジェクト固有のルールや指示をここに記載してください。

## プロジェクト概要

Claude Code を動かすためのルートリポジトリです。

## コーディング規約

- コミットメッセージは日本語または英語で記述する
- ファイル名はケバブケース（kebab-case）を使用する
- インデントはスペース 2 つを使用する

## よく使うコマンド

```bash
# Claude Code を起動
claude

# ヘルプを表示
claude --help
```

## 注意事項

- `ANTHROPIC_API_KEY` 環境変数または `.env` ファイルに API キーを設定すること
- シークレット情報（API キー等）をリポジトリにコミットしないこと
