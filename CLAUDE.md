# CLAUDE.md

このファイルは Claude Code がこのリポジトリで作業する際に参照するガイドラインです。

## プロジェクト概要

このリポジトリは Claude Code を活用した開発のためのルートリポジトリです。

## 開発ガイドライン

### コーディングスタイル

- コードには適切なコメントを付ける
- 関数・変数名はわかりやすい名前にする
- 変更は小さく、目的が明確になるようにする

### Git の使い方

- コミットメッセージは変更内容を簡潔に表す
- 機能ごとにブランチを作成する
- プルリクエストはレビューを経てマージする

### ファイル構成

- 新しいファイルを追加する場合は適切なディレクトリに配置する
- 不要なファイルは `.gitignore` に追加するか削除する

## よく使うコマンド

```bash
# Claude Code を起動する
claude

# 特定のファイルについて質問する
claude "src/main.ts を説明して"
```

## 注意事項

- 機密情報（APIキー、パスワードなど）はコードに直接書かない
- `.env` ファイルはリポジトリにコミットしない
